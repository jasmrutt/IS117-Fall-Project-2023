Internet Applications Semester Project

Spring 2023

Jasmin Rutter developed a website to advertise a fruit stand using php, html, and css. Her primary focus was on the navigation and architecture of information. She achieved this by creating multiple pages for various tasks. For example, the Home Page gave a brief description of the fruit stand's origin and showed various fruits the company sold. There was also a shipping page so the user's knew whether or not the dimensions of their packages were ok. Sql was used in the project to perform data migration (i.e. exporting information to be displayed). Javascript was also used for validation purposes.
