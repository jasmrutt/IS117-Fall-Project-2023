Internet Applications Semester Project

Spring 2023

I developed a website to advertise a fruit stand using php, html, and css. My primary focus was on the navigation and architecture of information. I achieved this by creating multiple pages for various tasks. For example, the Home page gave a brief description of the fruit stand's origin and showed various fruits the company sold, there was also a shipping page so the user's knew whether or not the dimensions of their packages were ok, a page to add and delete products, and a page to look at a table of all the products details (i.e. fruit, color, description, price). The table about the fruit's details was created through sql and performed data migration (i.e. importing new objects to the table, exporting information to be displayed on a website, deleting and adding objects in the table).
